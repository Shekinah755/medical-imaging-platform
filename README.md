 Medical Imaging & Radiology Systems Portfolio

This repository contains two Python-based healthcare systems built for medical imaging workflow management and image enhancement.


 1. Radiology Information System (RIS)

A desktop application that manages radiology workflow in a hospital environment.

 Features
- Patient registration and management
- Imaging scan recording (CT, MRI, X-ray)
- Radiology report creation and tracking
- User authentication with role-based access (Admin, Radiologist, Technician, Receptionist)
- Secure SQLite database backend
- Dashboard with system statistics

 Tech Stack
- Python
- Tkinter (GUI)
- SQLite (Database)
- SHA-256 (Password Security)

How to Run the Project
### Radiology Information System
```bash
cd "Radiology Information System"
python main.py
```
